# Agent Heartbeat

Agent heartbeat notes are periodic summaries, review prompts, and recurring project health checks.

## Suggested Use

- Daily: summarize active work and unresolved risks.
- Weekly: review architecture drift, test gaps, and release readiness.
- Monthly: review roadmap, tech debt, and governance updates.
