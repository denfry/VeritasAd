Project memory captures durable VeritasAd context that future agents should not rediscover from scratch.

- The backend is FastAPI with async video processing.
- The frontend is Next.js.
- Supabase, Celery, Redis, OpenTelemetry, Sentry, and structlog are part of the operating context.
