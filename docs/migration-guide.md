# Migration Guide

## For Existing Contributors

1. Read `AGENTS.md` and linked governance docs.
2. Use `.ai/workflows` templates for feature/refactor/release tasks.
3. Update `docs/changelog.md` and relevant docs on behavior/contract changes.
4. Include rollback notes for migration-sensitive changes.

## For Existing Branches

- Rebase and align PR descriptions with the new checklist requirements.
