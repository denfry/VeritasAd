# Reports

Раздел содержит **отчёты об экспериментах** исследовательского воркспейса VeritasAd: результаты прогонов моделей и пайплайнов, **ablation**-исследования (вклад отдельных компонентов), **error analysis** (разбор ошибок и типовых сбоев), а также отчёты о **annotation agreement** (согласованность разметки между аннотаторами, например по метрикам типа Cohen's κ / Krippendorff's α).

Каждый отчёт должен фиксировать воспроизводимый результат: постановку, конфигурацию, данные, метрики и выводы — так, чтобы его можно было сопоставить с другими прогонами.

## Структура

Раздел разбит по трекам:

- `master/` — отчёты трека **VeritasAd 2.0** (магистратура, вехи M1–M5).
- `phd/` — отчёты трека **VeritasAd 3.0** (PhD, вехи M1–M7).

## Именование файлов

Отчёты — это датируемые артефакты, поэтому имя файла строится по шаблону:

```text
YYYY-MM-DD-<milestone>-<slug>.md
```

Например:

```text
2026-09-15-m2-claim-extractor-ablation.md
2026-10-01-m3-annotation-agreement.md
```

Префикс `<milestone>` соответствует вехе (M1–M5 для master, M1–M7 для phd), а папка (`master/` или `phd/`) — треку.

Каждый отчёт начинается с YAML-фронтматтера и человекочитаемой строки статуса (см. шаблоны ниже).
