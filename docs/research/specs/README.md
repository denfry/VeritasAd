# Спеки (specs)

Раздел содержит **дизайн-спеки модулей и методов** проекта VeritasAd: формальные описания
архитектуры компонентов, алгоритмов, интерфейсов и методов верификации, которые ложатся в основу
последующей реализации и экспериментов.

## Структура

Раздел разбит по трекам:

- **`master/`** — спеки трека VeritasAd 2.0 (магистратура).
- **`phd/`** — спеки трека VeritasAd 3.0 (PhD).

## Именование файлов

Файлы спек именуются по общей конвенции датируемых артефактов:

```text
YYYY-MM-DD-<milestone>-<slug>.md
```

где фаза задаётся вложенной папкой (`master/` или `phd/`), а майлстоун — префиксом в имени файла.

## Содержимое

### master/

Уже содержит дизайн-спеку извлечения мультимодальных признаков для майлстоуна M2:

- [Multimodal feature-extraction design (M2)](./master/2026-06-16-multimodal-feature-extraction-design.md)

### phd/

Папка предназначена для спек верификации трека PhD, в частности:

- problem statement;
- source credibility model (модель доверия к источникам);
- verification metrics (метрики верификации);
- и др. спеки верификации.
