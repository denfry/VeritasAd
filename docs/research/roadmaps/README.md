# Roadmaps

Этот раздел содержит **два дорожных карты-источника истины** для проекта VeritasAd. Каждый документ описывает свою исследовательскую дорожку и служит единственным авторитетным источником по составу, нумерации и содержанию milestone'ов соответствующего трека.

Артефакты (планы, спецификации, отчёты, эксперименты, датасеты), привязанные к конкретным milestone'ам этих дорожных карт, кладутся в соответствующие разделы воркспейса: [`plans/`](../plans/), [`specs/`](../specs/), [`reports/`](../reports/), [`experiments/`](../experiments/), [`datasets/`](../datasets/). Имя датируемого артефакта начинается с префикса milestone'а (например, `2026-09-01-m2-claim-extractor-mvp.md`), а сам milestone определён здесь, в дорожной карте своего трека.

## Дорожные карты

### [VeritasAd 2.0 (магистратура)](./master-2.0.md)

Извлечение и классификация проверяемых рекламных утверждений. Milestone'ы **M1–M5**:

- **M1 — Research Foundation**
- **M2 — Claim Extraction MVP**
- **M3 — Dataset Pipeline**
- **M4 — Experiments**
- **M5 — Thesis Package**

Горизонт: 4 семестра.

### [VeritasAd 3.0 (аспирантура)](./phd-3.0.md)

Доказательная верификация утверждений. Milestone'ы **M1–M7**:

- **M1 — Verification Foundation**
- **M2 — Evidence Retrieval**
- **M3 — Source Credibility**
- **M4 — Evidence Relations**
- **M5 — Contradiction Analysis**
- **M6 — Verdict Generation**
- **M7 — Thesis Package**

Горизонт: 4 года.

## Примечание

Дорожные карты **не шаблонируются**: это уникальные документы-источники истины, а не экземпляры повторно используемого шаблона.
