# Литература

Раздел исследовательского воркспейса VeritasAd для работы с источниками: обзор литературы, проверка новизны и заметки по отдельным источникам.

## Назначение

Здесь собирается всё, что связано с научным контекстом проекта:

- **Обзор литературы** — систематизированный разбор связанных работ (related work), смежных подходов и существующих решений.
- **Проверка новизны** — фиксация того, чем VeritasAd отличается от уже опубликованного, и аргументация научного вклада (novelty verification).
- **Заметки по источникам** — индивидуальные конспекты по каждой прочитанной статье, книге или техническому отчёту.

## Содержание

- [Related Work & Novelty Verification (P0)](./related-work.md) — обзор связанных работ и проверка новизны.

## Заметки по источникам

Каждый источник конспектируется в отдельном файле. Имя файла строится по ключу из библиографии:

```text
<bib-key>.md
```

Например: `bekkerman2007.md`, `vaswani2017-attention.md`.

Заметки по источникам и обзор литературы оформляются с YAML-фронтматтером (как литературные заметки). Шаблон для новой заметки:

- [literature-note.template.md](../_templates/literature-note.template.md)
